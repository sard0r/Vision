This is an example of the usage of ROS and Camera usage together:

Following steps are neede to run the code: 

The disadvantage of the code is that currently we are seeking to make it compatible with the conda environment. The code will have environment problems when used together with anaconda.


1) Install ROS1 noetic following http://wiki.ros.org/noetic/Installation/Ubuntu

2) Install the requirements.txt file by the following command: pip install -r requirements.txt

3) Download the Rosbag file from the google drive: https://drive.google.com/drive/folders/1o06lsZGGVNWrrPUQIAk-KzyulQQTUApB?usp=share_link

4) Follow the uploaded youtube video to run the code: https://youtu.be/xkR2zoJjh9M

Enjoy!
